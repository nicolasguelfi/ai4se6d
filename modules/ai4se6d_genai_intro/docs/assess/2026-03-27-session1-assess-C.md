# ASSESS Report — Session 1 (Pathway C: Create)

**Date:** 2026-03-27
**Target:** Day 1, Blocks 1-2 (90 minutes total)
**Project:** ai4se6d_genai_intro (migrated from AI4SE-NG on 2026-03-31)

## Audience Profile
- **Who:** Software developers (intermediate level)
- **Prerequisites:** Solid experience in TypeScript, Python, Git
- **Group size:** 5-20 participants
- **Mindset:** Curious or hesitant about AI in software development
- **Language:** English (trainers speak French + English)
- **Context:** Professional training (DLH Luxembourg), April 9, 2026

## Content Objectives

### Block 1 — Introduction to Fundamentals of Generative AI (45 min)
- **O1:** Define AI vs. Generative AI — clear conceptual distinction
- **O2:** Map the GenAI tool landscape (ChatGPT, Gemini, Claude, Copilot, etc.)
- **O3:** Explain how LLMs work (transformers, tokens, training, inference) — accessible level
- **O4:** Address limitations, capabilities, and ethical considerations honestly

### Block 2 — Discovering VibeCoding / VibeEngineering (45 min)
- **O5:** Introduce VibeCoding as a paradigm shift in software development
- **O6:** Explain the philosophy: intent-driven development, AI as pair programmer
- **O7:** Position VibeCoding vs. VibeCoding within a broader VibeEngineering approach
- **O8:** Set the stage for the rest of the training (tools, methodology, practice)

## Design Intent

### R27 — Presentation Context
- Auditorium projection (10-20m distance)
- 16:9 ratio, dark theme
- Zoom 80% for readability
- ~45 min per block = ~12-15 slides per block

### R28 — Space Filling Strategy
- **MINIMALIST** — maximum white space, single focal point per slide
- Large images/illustrations as primary content carriers
- Text reduced to keywords, short phrases (3-7 words per bullet)
- One main idea per slide

### R29 — Design Profiles
- **Profile: Visual-First** — image dominates, text supports
- L1 (headline) + L2 (image | key points) layout preferred
- Full-bleed images when possible
- Icons/diagrams over bullet lists

### R30 — Profile Assignments
- Title slides: Full visual (image + title only)
- Concept slides: Image left | 3-4 key points right
- Transition slides: Single question or statement, centered

### R31 — Adaptive Calibration
- Font sizes: title 96pt, headline 80pt, body 48pt minimum
- Max 5 bullets per list, max 7 words per bullet
- 0-1 emoji per slide
- Color palette: primary (#7AB8F5), accent (#2EC4B6), highlight (#F39C12)
