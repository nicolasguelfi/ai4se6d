# Design Guideline — ai4se6d-collection

guideline: minimalist-visual

## Context
- Collection hub / landing page for the AI4SE 6D training
- Dark theme, wide layout
- Displays project cards linking to individual training modules
- Should convey clarity, professionalism, and easy navigation

## Overrides
None — minimalist-visual applies to all blocks.

## Layout Rules
- Cards: centered grid, consistent spacing
- Title: large, primary color, centered
- Description: concise, muted text
- No heavy imagery — emojis and clean typography only
