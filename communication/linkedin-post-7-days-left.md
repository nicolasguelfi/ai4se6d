## Option GIF/animation

Pour une courte animation (3-5 secondes), voici des outils gratuits :

1. **Canva** (canva.com) — Créer un post animé LinkedIn directement :
   - Template "LinkedIn Post"
   - Fond sombre avec animation de compte à rebours "7 → 0"
   - Export en MP4 (LinkedIn accepte les vidéos courtes)

2. **ScreenToGif** (screentogif.com) — Si tu veux capturer un GIF de ton terminal :
   - Enregistre-toi en train de lancer `claude` dans un terminal
   - Claude qui génère du code en direct (3-5 secondes)
   - Très authentique et accrocheur pour un public dev

3. **LottieFiles** (lottiefiles.com) — Animations vectorielles gratuites :
   - Cherche "coding", "AI", "countdown"
   - Exporte en GIF ou MP4

### Ma recommandation

**L'image statique avec le prompt minimaliste** est le meilleur ratio effort/impact pour LinkedIn. Les posts avec une image simple et sombre performent mieux que les GIFs (qui ne se lancent pas toujours automatiquement sur mobile).

Si tu veux un GIF, la capture de terminal avec Claude Code en action est de loin le plus authentique et le plus accrocheur pour ton audience de développeurs.
